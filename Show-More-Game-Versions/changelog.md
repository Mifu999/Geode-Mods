# Changelog

## v1.0.0
- Initial release of Show More Client Versions
- Display Geometry Dash version
- Display Geode Loader version
- Display Cocos2d version
- Display Operating System with version detection (Windows, macOS, Android, iOS, Linux)
- Display CPU Architecture (x64, x86, ARM64, ARM32)
- Display detailed mod statistics (enabled, disabled, total counts)
- 5 position presets: Top Left, Top Right, Bottom Left, Bottom Right, Custom
- Custom positioning with precise X/Y coordinates
- Adjustable text scale (0.1x to 1.0x)
- Opacity control (0 to 255)
- Single line or multi-line display modes
- Individual toggles for each information piece
- Debug logging support
- All system information features disabled by default for clean initial experience
