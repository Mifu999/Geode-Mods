# Show More Client Versions

A comprehensive mod that displays detailed game, system, and mod information on the main menu of Geometry Dash.

### Game Information
- Geometry Dash version
- Geode Loader version
- Cocos2d engine version

### System Information (optional)
- Operating System with version detection (Windows 10/11, macOS, Android, iOS, Linux)
- CPU Architecture (x64, x86, ARM64, ARM32)
- Detailed mod statistics showing enabled, disabled, and total mod counts

## Customization

This mod offers extensive customization options:

- **5 Position Presets**: Top Left/Right, Bottom Left/Right, or Custom coordinates
- **Precise Positioning**: Set exact X/Y coordinates for custom placement
- **Adjustable Scale**: Resize text from 10% to 100% of default size
- **Opacity Control**: Make the text more or less transparent
- **Flexible Layout**: Display information in a single line or multiple lines
- **Selective Display**: Toggle each information piece individually

All system information features are optional and disabled by default, allowing you to choose exactly what you want to see.
