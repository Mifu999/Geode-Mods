# Show More Client Versions

A comprehensive Geometry Dash mod that displays detailed game, system, and mod information on the main menu with extensive customization options.

## Features

### Version Information
- **Geometry Dash version** - Display the current GD version
- **Geode Loader version** - Display the Geode framework version
- **Cocos2d version** - Display the Cocos2d engine version

### System Information
- **Operating System** - Shows your OS (Windows 10/11, macOS, Android, iOS, Linux)
- **CPU Architecture** - Displays your processor architecture (x64, x86, ARM64, ARM32)
- **Mod Statistics** - Shows detailed mod count (enabled, disabled, total)

### Customization Options
- **Flexible Positioning** - Choose from 5 preset positions:
  - Top Left
  - Top Right
  - Bottom Left
  - Bottom Right
  - Custom (with precise X/Y coordinates)
- **Adjustable Scale** - Resize the text from 0.1x to 1.0x
- **Opacity Control** - Set transparency from 0 (invisible) to 255 (fully opaque)
- **Display Format** - Single line or multi-line layout
- **Toggle Individual Items** - Show or hide any information piece
