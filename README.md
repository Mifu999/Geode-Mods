## Installation

Install [Geode](https://geode-sdk.org/), then download mods from the [releases page](https://github.com/Mifu999/Geode-Mods/releases).

## Building
```bash
# Clone the repository
git clone https://github.com/Mifu999/Geode-Mods
cd Geode-Mods

# Navigate to the mod you want to build
cd Show-More-Game-Versions

# Build with Geode CLI
geode build
```
